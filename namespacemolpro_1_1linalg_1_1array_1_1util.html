Starting from id object checks that each link in the path exists.

E.g. if id is file and path="/g1/g2/g3" it checks that "/g1", "/g1/g2", and "/g1/g2/g3" all exist.

Parameters
idhdf5 object id
pathpath to the link relative to the id. If id is a file object, should include root "/"
Returns
>0 if link exists, 0 if it doesnt, <0 if it failed or path is empty.

Find duplicates references to x and y arrays and store unique elements in a separate vector.

Vector of scalars is returned complete without truncation. The register becomes a vector of pairs, first element of a pair containing index of x array in the vector of unique x arrays, the second is for y arrays.

Returns
