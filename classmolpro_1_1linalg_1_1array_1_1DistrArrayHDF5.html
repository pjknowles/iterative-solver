Detailed Description

Distributed array storing the buffer on disk using hdf5.

Warning
Before using operations that require access to the file, the hdf5 file must be opened. To avoid corruption on accidental termination the file must be closed when the data is no longer needed.

Use cases

Backing up:

auto backup(const DistrArray &current_solution, std::shared_ptr<PHDF5Handle> backup_file){
return DistrArrayDisk{current_solution, backup_file};
}

Create a disk array with a file assigned and default distribution.

By default array is distributed uniformly with any remainder spread over the first processes so their size is larger by 1.
